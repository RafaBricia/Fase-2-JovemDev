Principais conceitos


  1. Variáveis e Tipos de Dados:
    2.

    Variáveis: São espaços de armazenamento para dados na memória. Cada variável possui um nome
    e um tipo de dado associado.

    Tipos de Dados: Representam os diferentes tipos de valores que uma variável pode armazenar, como
    inteiros, números de ponto flutuante, caracteres, etc.


  2. Operadores:
    3.

    Aritméticos: Realizam operações matemáticas básicas (adição, subtração, multiplicação, divisão).

    Relacionais: Comparação entre valores (igualdade, maior que, menor que, etc.).

    Lógicos: Operações lógicas como AND, OR, NOT.


  3. Controle de Fluxo:

    Condicionais: Estruturas como if, else, switch, que permitem a execução condicional de blocos de código.

    Loops (Laços): Como for, while, do-while, para repetir a execução de um bloco de código enquanto uma
    condição é atendida.


  4. Funções:

    Blocos de código que podem ser chamados/executados a partir de outros lugares do programa. Permitem
    modularizar o código e evitar repetições.


  5. Arrays (Vetores):

    Arrays: Estruturas que armazenam coleções de dados do mesmo tipo. Os elementos são acessados por
    meio de índices.


  6. Estruturas de Dados:

    Listas, Filas, Pilhas: Estruturas que organizam e armazenam dados de maneiras específicas, facilitando
    operações como inserção, remoção e busca.
