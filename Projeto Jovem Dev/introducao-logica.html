O que é ser um programador?


Um programador escreve códigos simples e complexos por meio de linguagens de programação como
Java, Javascript, Python, C#, e muitas outras. Tudo isso acontece com o objetivo de corrigir possíveis erros,
desenvolver soluções para problemas, criar comandos e novas funcionalidades para softwares, programas e sistemas.

Dentre as diversas funções, eles precisam testar os códigos e assegurar que estão em pleno funcionamento. Podem
criar aplicativos de celular, redes sociais e sites. Esse profissional também é responsável por propor melhorias conforme
a necessidade dos usuários.


O que é lógica de programação?


A lógica de programação é uma das habilidades mais requeridas na vida de uma pessoa desenvolvedora.

É a maneira como são orientadas as ações executadas por algoritmos. Toda programação é composta por
uma sequência lógica de códigos descritos, e por meio deles, executar os comandos atribuídos. Nesse sentido,
as pessoas que programam
que tem responsabilidade de entender a lógica de execução dos códigos, e traduzir para a linguagem necessária
para que a máquina entenda.

Os algoritmos dessa lógica são uma sequência de regras, ou mesmo comportamentos, que vão executar
determinada operação para que funcione adequadamente. Assim, para que um algoritmo possa operar sem
incoerências no código. É essencial escrever cada comando e seus respectivos dados de forma clara e objetiva.

(Introdução a Algoritmos - Gustavo Guanabara)
