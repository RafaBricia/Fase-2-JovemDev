Matrizes


Uma matriz é uma estrutura bidimensional que armazena elementos organizados em linhas e colunas.
Em lógica de programação, pode ser vista como um vetor de vetores.


  • Matrizes são definidas utilizando colchetes aninhados [[]] para representar as linhas e colunas.

  • Os elementos de uma matriz são acessados usando índices duplos, indicando a linha e a coluna.

  • O número de linhas e colunas de uma matriz pode ser obtido usando as funções len() e len()[0], respectivamente.

  • Loop aninhado for é frequentemente utilizado para percorrer os elementos de uma matriz.

  • Assim como em vetores, você pode modificar, adicionar ou remover elementos de uma matriz.
