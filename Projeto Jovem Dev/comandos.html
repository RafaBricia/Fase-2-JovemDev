Comando de entrada e saída


  • Comando de entrada

    Leia: A instrução Leia é utilizada quando se deseja obter informações do usuário por meio do teclado,
    ou seja, é um Comando de Entrada de Dados.

    Sintaxe:
    Leia(Variavel);


  • Comando de saída

    Escreva: A instrução Escreva é utilizada para mostrar informações na tela do computador, ou seja, é um
    Comando de Saída de Dados.

    Sintaxe:
    Escreva(Variavel);
